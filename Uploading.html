Assumptions

  • Initially, we need to consider that every input dataset should be preprocessed.
  • Files should be a text file(.txt extension) of CSV format.
  • User should be aware of type of models to be used.
  • Here, we will consider the last column as a dependent variable (i.e, we say y).
  • Remaining columns are considered as independent (x).
  • Moreover, we should not consider the heading of our files, we should ignore them.
  • Parameters Learning rate and epochs values can either be set as default or even the user can give his customized values.
  • Moreover, user can play with the parameters if not satisfied with the predicted values.
  • Feature-Normalisation is enabled where ever required.

Supervised

In supervised learning, we are given a data set and already know what our correct output should look like, having the idea that there is a relationship between the input and the output.

UnSupervised

Unsupervised learning allows us to approach problems with little or no idea what our results should look like. We can derive structure from data where we don't necessarily know the effect of the variables.
